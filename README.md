# Enquetes - Teste SIGNOWEB

## Ferramentas

Foi utilizado PHP, MySQL, Docker, Javascript e CSS.

## Instalação com Docker

```
docker-compose up --build
```

Utilize o script `banco_de_dados.sql` para criar as tabelas e inserir dados de exemplo.

Obs: Caso não utilize o docker, descomente a primeira linha do script para criar o banco de dados.

Entre em:

```
http://localhost
```
